# osu!hit-effect generator
This is a generator which combines an effect (a wanted hit feedback) with the hitX files to both have an animated hit effect and a responsive feedback. This is especially useful when playing low AR+HD, which is why i initially created this program

## Set-up
1. Run the `setup.py`
2. Edit the `config.ini`

> [!IMPORTANT]
> **ALL** paths in config must be set to an image before running. This will be fixed later, but too lazy to fix as of now. 

## Running
1. Run the `main.py` after setting up
